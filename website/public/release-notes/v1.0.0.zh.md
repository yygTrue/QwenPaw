**多 Agent 系统**

- **全局 LLM 流控器**：QPM（每分钟请求数）滑动窗口限流、信号量并发控制，以及收到 429 响应时的全局暂停协调机制（带随机抖动避免雷鸣重试） ([#2282](https://github.com/agentscope-ai/CoPaw/pull/2282))
